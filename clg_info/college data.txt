5. Course Syllabi
Subject: Data Science (Code: BCS-503)
Program: B.Sc. Computer Science (Third Year), VI Semester 


Objectives: Learn fundamental concepts/operations, work on algorithms, and explore current applications.


Outcomes: Review fundamentals, evaluate techniques for analysis, develop algorithms, and evaluate tools.

Units:

Unit I: Introduction to Data Science (Data Mining, Visualization, Software Engineering trends) 

Unit II: Database & Warehousing, AI & ANN basics, Statistics, Hypothesis techniques 

Unit III: Computational techniques, Machine Learning, Big Data, Research Methodology 

Unit IV: Applications, Future importance, Programming paradigms 

Unit V: Data Mining vs. Data Science, Predictive Analytics, Exploratory Data Analysis 

Unit VI: Optimization, Roles/Responsibilities, Data lifecycle, Hadoop Integration with R 


Reference Books: Data Smart (Foreman), Fundamentals of mathematical statistics (Gupta/Kapoor), Database Design (C.J. Date), Data Mining (Dunham).

Subject: Software Testing - Elective (Code: BCS-504A)
Program: B.Sc. Computer Science (Third Year), VI Semester 


Objectives: Develop testing execution skills, understand techniques, enhance QA skills.


Outcomes: Learn development methods, apply testing techniques, evaluate costs, implement types of testing.

Units:

Unit I: Quality concepts (McCall’s/ISO 9126 factors, Cost of Quality, QA) 

Unit II: Software Quality Assurance (Reviews, Reliability, QA Plan) 

Unit III: Software Testing Strategies (Unit, Integration, Validation, System Testing, Debugging) 

Unit IV: Testing Application (White/Black Box, Basic Path, Control Structural) 

Unit V: WebApps for Testing (Content, UI, Navigation, Security) 

Unit VI: Product Metrics (Requirements, Design, Source code, Testing metrics) 


Reference Books: Software Engineering – A Practitioner's approach (Roger S. Pressman, 5th & 6th Editions), Software Testing Concepts and Tools (NageswaraRoo).
